# Database Files

This directory contains all database-related files:

## Migration Files
- `database_setup.sql` - Main database schema with tables, constraints, and sample data
- `database_migration.sql` - Core migration file
- `database_chat_migration.sql` - Chat feature database schema
- `database_time_tracking_migration.sql` - Time tracking feature database schema
- `database_user_management.sql` - User management feature database schema
- `database_user_password_migration.sql` - User password management schema

## Debug Files
- `debug_chat_issue.sql` - Debugging queries for chat functionality
